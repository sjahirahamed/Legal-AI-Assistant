Developed a Retrieval-Augmented Generation (RAG) system that maps real-world crime scene descriptions to the most relevant Indian Penal Code (IPC) sections.

Implemented semantic search using Sentence Transformers and ChromaDB to retrieve top-k IPC laws for a given case.

Integrated Google Gemini API to generate legal explanations and rationales, citing applicable IPC sections.

Delivered a prototype capable of assisting law students and professionals in quickly identifying relevant laws.

Tech Stack: Python, ChromaDB, Sentence Transformers, Google Gemini (google-genai), R
